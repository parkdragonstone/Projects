# Bike (EMG)
약 20분 간의 자전거 주행에서 근전도로 근육 신호를 측정하고 피로도를 측정함.
- 측정 도구 : Delsys Wireless EMG
- 부착 위치 : Rectus Femoris (RF), Vastus Lateralis (VL), Biceps Femoris (BF),      Gasctrocnemuius (GM)
- 독립 변인 : 안장 위치, 높이
- 종속 변인 : 중앙 주파수, 적분근전도 등
- 통계 : Repeated Measures ANOVA
